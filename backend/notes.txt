CREATING EXPRESS APP(REST API)

    Create empty folder
    Create package.json file
        npm init
    Create entry point
        new app.js file
    Install
        npm install express
    Create server
        import server
        assign port number
    Define routes(req handler) of API
        app.httpMethod("path",req handler)
    User api
        getallusers
            client url : http://localhost:3000/getallusers
        getuser/id
            client url : http://localhost:3000/getuser/1
        createuser
            client url : http://localhost:3000/createuser
            provide json data
        updateuser
            client url : http://localhost:3000/updateuser
            provide json data
        deleteuser
            client url : http://localhost:3000/deleteuser/id


MIDDLEWARE-->define middleware before calling it 
    client->m1(next)->m2(next)->server

    nothing but functions that act as checks btw request and api route
        (request, response, next) =>{}
    can execute any code
    can send response to client

    USING MIDDLEWARE

        execute middleware for each req
            app.use(middleware)
        execute for a specific req
            app.httpMethod("path",middleware,handler)
        execute for a specific path
            app.use("path",middleware)

    HANDLING INVALID PATHS-->called at last

    html page is returned to the client instead of json object
    middlewares could be used to send appr. messages


    ERROR HANDLING MIDDLEWARES-->called at last

    (error,request,response,next)
    error.message
